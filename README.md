# AI-Agent-Booking-system
# Booking System Chatbot 

A conversational AI booking system using **Dialogflow ES/CX**, **FastAPI** backend, and **MySQL** database.

## Features
- Service booking via chat (Dialogflow)
- FastAPI backend to handle bookings
- MySQL database to store services and bookings


## Tech Stack
- Dialogflow ES/CX
- FastAPI (Python)
- MySQL
